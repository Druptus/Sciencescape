# Cheatsheet:



## Terminal (File Explorer)

| Command | What it does |
|:---:|:---:|
| ls | list files and folders |
| cd | enter a folder |
| vim file.py | open a file for editing* |
| python3 file.py | run a python file* |
\* **file.py is a placeholder name!**



## Vim (File Editor)

| Command | What it does |
|:---:|:---:|
| i | start editing |
| Esc | stop editing |
| :w | save file contents* |
| :wq | save and exit file* |
| :q! | quit WITHOUT saving (undo everything)* |
\* **Only works when not editing!**
